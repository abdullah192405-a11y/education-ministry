# 🤖 توليد الأسئلة والألعاب بالذكاء الاصطناعي - AI Question Generator

## ✨ نظرة عامة

تم إضافة ميزة جديدة **ثورية** تستخدم **Gemini 2.5 Flash** من Google لتوليد الأسئلة والألعاب التفاعلية تلقائياً من ملفات PDF!

---

## 🎯 الهدف

تسهيل عمل المعلمين بتوليد أسئلة وألعاب تعليمية احترافية بضغطة زر واحدة، بدلاً من إنشائها يدوياً واحدة تلو الأخرى.

---

## 🚀 الميزات

### 1️⃣ **رفع ملف PDF**
- رفع أي ملف PDF تعليمي (كتاب، ملخص، ورقة عمل)
- دعم ملفات بأحجام مختلفة
- معاينة اسم الملف والحجم

### 2️⃣ **التحكم الذكي بالتوليد**
المعلم يكتب prompt محدد مثل:
- "أريد 10 أسئلة عن الوحدة الأولى: الخلية ومكوناتها"
- "ولّد 5 ألعاب تفاعلية عن الدرس الثاني"
- "أسئلة صعبة عن الصفحات 15-25"
- "اخلط بين الأسئلة والألعاب عن موضوع الجهاز الهضمي"

### 3️⃣ **أنواع متعددة من الأسئلة والألعاب**
الذكاء الاصطناعي يُولّد:

#### 📝 الأسئلة:
- ✅ اختيار متعدد (Multiple Choice)
- ✅ صح وخطأ (True/False)
- ✅ سؤال وجواب (Q&A)
- ✅ أعرف/لا أعرف (Know/Don't Know)
- ✅ ترتيب (Order Questions)

#### 🎮 الألعاب:
- ✅ مطابقة (Matching)
- ✅ تصويب (Shooting)
- ✅ عجلة الحظ (Wheel Spin)
- ✅ ألغاز (Puzzle)

### 4️⃣ **معالجة ذكية**
- قراءة محتوى PDF
- فهم السياق والموضوع
- توليد أسئلة ذات صلة
- تنسيق JSON صحيح
- إضافة النقاط والوقت تلقائياً

### 5️⃣ **مراجعة وتعديل**
- الأسئلة المولدة تُضاف للقائمة
- يمكن مراجعتها واحدة تلو الأخرى
- تعديل أي سؤال حسب الرغبة
- حذف ما لا يناسب
- إعادة ترتيب

## 🔧 الإعداد والتكوين

### الخطوة 1: الحصول على مفتاح Gemini API

1. اذهب إلى: https://makersuite.google.com/app/apikey
2. سجل الدخول بحساب Google
3. اضغط "Create API Key"
4. انسخ المفتاح

### الخطوة 2: إضافة المفتاح في المشروع

1. انشئ ملف `.env` في جذر المشروع (إذا لم يكن موجوداً):
```bash
# في نفس المجلد مع package.json
```

2. أضف المفتاح:
```bash
VITE_GEMINI_API_KEY=AIzaSyD...your_actual_key_here
```

3. **مهم**: تأكد أن `.env` موجود في `.gitignore`:
```
.env
.env.local
```

### الخطوة 3: إعادة تشغيل السيرفر
```bash
npm run dev
```

## 🧠 كيف يعمل الذكاء الاصطناعي؟

### سير العمل:

```
1. المعلم يرفع PDF
        ↓
2. النظام يقرأ محتوى PDF
        ↓
3. يبني prompt شامل للـ AI:
   - محتوى PDF
   - طلب المعلم
   - أنواع الأسئلة المتاحة
   - تنسيق JSON المطلوب
        ↓
4. يرسل لـ Gemini 2.5 Flash
        ↓
5. Gemini يحلل النص ويفهم الموضوع
        ↓
6. يولد أسئلة وألعاب ذكية
        ↓
7. يرجع JSON منسق
        ↓
8. النظام يتحقق من الصحة
        ↓
9. يضيف الأسئلة للقائمة
        ↓
10. المعلم يراجع ويحفظ
```

### مثال Prompt المُرسل للـ AI:

```
أنت مساعد ذكي متخصص في إنشاء أسئلة تعليمية...

المحتوى من PDF:
[محتوى الـ PDF كاملاً]

طلب المعلم:
"أريد 10 أسئلة عن الوحدة الأولى"

يرجى إنشاء أسئلة بالتنسيق التالي:
[
  {
    "type": "multiple_choice",
    "question": "...",
    "options": [...],
    "correctAnswer": 0,
    ...
  }
]

أنواع متاحة: [قائمة كاملة بالأنواع]
```

## 🐛 استكشاف الأخطاء

### "لم يتم تكوين مفتاح Gemini API"
```
السبب: المفتاح غير موجود في .env
الحل:
1. تأكد من وجود VITE_GEMINI_API_KEY في .env
2. أعد تشغيل npm run dev
```

### "خطأ في API: 401 Unauthorized"
```
السبب: المفتاح غير صحيح أو منتهي
الحل:
1. راجع المفتاح في .env
2. جرب مفتاح جديد من Google AI Studio
```

### "لم يتم توليد أي محتوى"
```
السبب: prompt غير واضح أو PDF فارغ
الحل:
1. اكتب prompt أكثر تحديداً
2. جرب PDF مختلف
3. راجع console للأخطاء
```

### "فشل في تحليل الأسئلة"
```
السبب: JSON غير صحيح من AI
الحل:
1. جرب مرة أخرى (قد يكون خطأ عابر)
2. راجع console.log للـ response
```
