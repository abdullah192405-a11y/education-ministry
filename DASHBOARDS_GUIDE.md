## 1️⃣ **لوحة تحكم الطالب** (StudentDashboard.tsx) ✅ **تم إنشاؤها**

### الموقع:
```
src/pages/dashboard/StudentDashboard.tsx
```

### الميزات الرئيسية:

#### 📈 **نظرة عامة (Overview)**
- إحصائيات الطالب العامة:
  - **الدروس المكتملة**: عدد الموضوعات التي أكملها
  - **متوسط النتائج**: النسبة المئوية للدرجات
  - **ساعات الدراسة**: إجمالي الوقت المستثمر
  - **الشارات المحققة**: عدد الإنجازات

- **بطاقة الترحيب**:
  - تحية شخصية بالاسم
  - عرض السلسلة اليومية (Streak)
  - زر "تصفح الدروس" → يوجه إلى `/grades`

- **تقدم المواد**:
  - عرض التقدم لكل مادة دراسية
  - شريط تقدم ملون لكل مادة
  - النسبة المئوية للإكمال
  - متوسط النتائج للمادة

- **آخر الدروس**:
  - قائمة بآخر 3 دروس تم إكمالها
  - روابط مباشرة: `/grade/{gradeId}/subject/{subjectId}/topic/{topicId}`
  - عرض النتيجة والتاريخ

- **النشاط الأسبوعي**:
  - رسم بياني لنشاط الطالب خلال الأسبوع
  - استخدام `Progress` component

#### 📚 **المواد الدراسية (Subjects)**
- عرض جميع مواد الصف الحالي للطالب
- بطاقات تفاعلية لكل مادة:
  - أيقونة المادة الملونة
  - عدد الدروس
  - نسبة الإكمال
  - متوسط النتائج
- روابط مباشرة: `/grade/{gradeId}/subject/{subjectId}`

#### 📜 **سجل الدروس (History)**
- عرض تفصيلي لجميع الدروس المكتملة
- لكل درس:
  - اسم الموضوع والمادة
  - النتيجة بألوان مميزة (أخضر/أصفر/أحمر)
  - التاريخ
  - زر "إعادة الدرس"
  - زر "مشاركة"

#### 🏆 **الشارات (Badges)**
- عرض جميع الشارات المحققة وغير المحققة
- Grayscale للشارات غير المحققة
- علامة ✓ للشارات المحققة
- أمثلة:
  - 🏆 مثالي - أجبت على جميع الأسئلة بشكل صحيح
  - ⚡ البرق - إجابة سريعة
  - 🔥 متسلسل - 7 أيام متتالية
  - 📚 العالِم - أكثر من 90%

#### ⚙️ **الإعدادات (Settings)**
- معلومات الحساب (الاسم، البريد)
- إعدادات الإشعارات:
  - إشعارات الدروس الجديدة
  - إشعارات الشارات
  - تذكيرات السلسلة اليومية
  - تحديثات المواد

### بيانات تجريبية:
```typescript
const mockStudentData = {
    name: "أحمد الحربي",
```

## 2️⃣ **لوحة تحكم المعلم/القناة** (ChannelDashboard.tsx) ⚠️ **تحتاج تحديث بسيط**

### الموقع:
```
src/pages/dashboard/ChannelDashboard.tsx
```

### التحديثات المطلوبة:

#### ما يعمل حالياً:
- ✅ إدارة المحتوى (CRUD Operations)
- ✅ إنشاء تحديات جماعية
- ✅ مراقبة التحديات النشطة
- ✅ إحصائيات المشاهدات والمشاركين
- ✅ محرر المحتوى الكامل (ContentEditor)

#### ما يحتاج تحديث:
1. **تغيير المصطلحات**:
   ```typescript
   // قديم
   "المحتويات التعليمية" → "الدروس التعليمية"
   "إضافة محتوى جديد" → "إضافة درس جديد"
   
   // جديد
   "الموضوع: حروف الهجاء" (بدلاً من "المحتوى: حروف الهجاء")
   ```

2. **تحديث الروابط**:
   ```typescript
   //  قديم
   `/channel/${channelId}/content/${contentId}/challenge`
   
   // جديد
   `/grade/${gradeId}/subject/${subjectId}/topic/${topicId}/challenge`
   ```

3. **عرض المواد بدلاً من القنوات**:
   ```typescript
   // يمكن عرض المعلم كـ "مسؤول مادة"
   const mockTeacherData = {
       gradeId: 1,
       subjectId: 1, // اللغة العربية
       name: "أ. فاطمة الحربي",
       topics: [...] // الدروس التي أنشأتها
   };
   ```

### الميزات الحالية (التي تعمل):

#### 📊 **نظرة عامة**
- إحصائيات القناة/المادة
- التحديات النشطة (Admin + Community)
- أفضل المحتويات
- آخر المشاركين

#### 📝 **إدارة المحتوى**
- **CRUD كامل**:
  - إنشاء درس جديد
  - تعديل درس موجود
  - حذف درس
  - نشر/إخفاء درس

- **محرر محتوى متقدم** (ContentEditor):
  - إضافة media (فيديو، صور، نصوص)
  - إضافة أسئلة اختبار
  - إضافة عناصر التحدي (Challenge Items):
    - MCQ (اختيار متعدد)
    - True/False
    - Matching Game
    - Wheel of Fortune
    - Target Shooter

#### 🎮 **إدارة التحديات**
- **إنشاء تحديات رسمية**:
  - اختيار الدرس
  - توليد PIN تلقائي
  - التوجه لوضع المضيف

- **مراقبة التحديات**:
  - تحديات رسمية (Admin)
  - تحديات المجتمع (User-created)
  - نسخ PIN
  - التحكم / المراقبة
  - إنهاء التحدي

#### 📈 **الإحصائيات (Analytics)**
- عدد المشاهدات
- عدد التحديات
- متوسط النتائج
- معدل الإكمال
