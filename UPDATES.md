# تحديث المنصة - وزارة التربية والتعليم السعودية

## 📋 ملخص التحديثات

تم تحويل المنصة من نظام القنوات العام إلى نظام تعليمي متخصص لوزارة التربية والتعليم السعودية.

## 🏗️ الهيكل الجديد

### المستويات التعليمية:
```
الصف الدراسي (Grade)
    ↓
المادة الدراسية (Subject)
    ↓
الموضوع التعليمي (Topic)
    ↓
المحتوى (Media - فيديو، صور، نصوص)
    ↓
التحدي (Challenge)
```

### الهيكل القديم (للمقارنة):
```
القناة (Channel)
    ↓
المحتوى (Content)
    ↓
التحدي (Challenge)
```

## 📁 الملفات المُنشأة

### 1. نظام البيانات الجديد
- **`src/data/educationData.ts`** - هيكل البيانات التعليمية الكامل
  - `Grade` - الصف الدراسي (ابتدائي، متوسط، ثانوي)
  - `Subject` - المادة الدراسية (رياضيات، علوم، لغة عربية، إلخ)
  - `Topic` - الموضوع التعليمي
  - Helper functions لاسترجاع البيانات

### 2. صفحات جديدة
- **`src/pages/Grades.tsx`** - صفحة عرض جميع الصفوف الدراسية
- **`src/pages/GradeDetail.tsx`** - صفحة تفاصيل الصف (عرض المواد)
- **`src/pages/SubjectView.tsx`** - صفحة عرض المادة (عرض المواضيع)
- **`src/pages/TopicView.tsx`** - صفحة عرض الموضوع (المحتوى + التحديات)

### 3. مكونات محدثة
- **`src/components/landing/GradesSection.tsx`** - قسم الصفوف في الصفحة الرئيسية
- **`src/components/landing/HeroSection.tsx`** - تحديث النصوص للوزارة
- **`src/pages/Index.tsx`** - استخدام GradesSection بدلاً من ChannelsSection

## 📊 البيانات التعليمية المتوفرة

### الصفوف:
1. **الصف الأول الابتدائي**
   - اللغة العربية (حروف الهجاء، الحركات والتنوين)
   - الرياضيات (الأرقام 1-10، الجمع البسيط)
   - العلوم (الحواس الخمس)

2. **الصف الثالث الابتدائي**
   - الرياضيات (جدول الضرب)

3. **الصف الأول المتوسط**
   - العلوم (الخلية ومكوناتها)

## 🎯 مسار الاستخدام الجديد

1. المستخدم يدخل الصفحة الرئيسية
2. يختار "استكشف الصفوف"
3. يختار صفه الدراسي (مثلاً: الصف الأول الابتدائي)
4. يختار مادة (مثلاً: اللغة العربية 📖)
5. يختار موضوع (مثلاً: حروف الهجاء العربية)
6. يشاهد المحتوى (فيديوهات، صور، نصوص)
7. ينضم للتحدي (فردي أو جماعي)

## 🔗 الروابط الجديدة

- **الصفحة الرئيسية**: `/`
- **جميع الصفوف**: `/grades`
- **تفاصيل صف**: `/grade/1`
- **مادة دراسية**: `/grade/1/subject/1`
- **موضوع تعليمي**: `/grade/1/subject/1/topic/1`
- **التحدي**: `/grade/1/subject/1/topic/1/challenge`
